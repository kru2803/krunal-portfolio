Featured Project

EduRAG

// AI-Powered Education Knowledge Assistant using RAG

A production-grade Retrieval-Augmented Generation chatbot built from scratch — not using OpenAI's paid APIs, but running a local Mistral LLM through Ollama. EduRAG ingests educational documents, chunks and embeds them into ChromaDB, and uses semantic similarity search to retrieve the most relevant context before generating accurate, grounded answers. No hallucinations from a knowledge vacuum.

  • Built a full RAG pipeline: document ingestion → embedding → vector storage → semantic retrieval → LLM generation
  • Integrated Mistral LLM via Ollama for fully local, cost-free inference with context-aware responses
  • Implemented semantic search using Sentence Transformers (all-MiniLM-L6-v2) and ChromaDB as the vector store
  • Deployed an interactive Streamlit interface — clean, usable, demo-ready
